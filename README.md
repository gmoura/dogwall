# dogwall

Demo: https://dogwall.gmoura.now.sh/

## Run local and install

#### Install

`$ yarn`

#### Run local

`yarn start`

## Stack

- Yarn
- React
- Redux
- Material-UI
- React Hooks
- Deploy on https://zeit.co/now
- Husky
- Eslint
- `react-intl`, `react-router-dom`, `redux-thunk`, `prettier`
- dotenv
